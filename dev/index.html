Installation

You can install this package with:

using Pkg
Pkg.add(url="https://github.com/AR102/Interpreting-EEG-with-AI", rev="refactor")

Basic Usage

Include the package in your code with

using BCIInterface

To start collecting EEG data, first create an EEGBoard using the instructions in Supported Boards.

An offline example for trying out the rest of the framework would be

board = MCP3208("UselessBoard", 8, online = false)

Then create an Experiment](@ref) with it:

experiment = Experiment(board, "NameOfMyExperiment")

Now, say you want to later let an AI predict what colour a test person is thinking of. For this, you want the AI to classify the data using three categories: The person thinking of red, blue and yellow.

A possible setup would be to first make the test person think of red for 5 seconds (by telling them to or showing it on a screen) and start gathering data at the same time using gather_data!

gather_data!(experiment, 5, tags=[:red])
160702

Then, you repeat the same for yellow and blue, replacing [:red] by [:yellow] and [:blue] respectively.

To access the data, you can either use the Experiment object if you're still in the same session

Or you can load it from the file where it should have been automatically saved:

data = load_data("NameOfMyExperiment", "RawData")

Now you can start to train an AI using your collected data.

First, you need to create a ModelData object. This stores the data in a format usable by an artificial neural network.

In this format, every category of inputs that we want to classify is assigned an output which the model should return whenever it recognizes this category.

This is done via a Dict where each key is the label of all inputs that belong into one category and its value the output.

In our case, this could be something like

outputs = Dict(
    :red => [1.0, 0.0, 0.0],
    :yellow => [0.0, 1.0, 0.0],
    :blue => [0.0, 0.0, 1.0]
)
Dict{Symbol, Vector{Float64}} with 3 entries:
  :red    => [1.0, 0.0, 0.0]
  :yellow => [0.0, 1.0, 0.0]
  :blue   => [0.0, 0.0, 1.0]

And now to create the ModelData object:

modeldata = ModelData(data, outputs)
ModelData(DataLoader(::NamedTuple{(:inputs, :outputs), Tuple{Array{Float32, 3}, Matrix{Float32}}}, shuffle=true, batchsize=5), Dict{Symbol, Array{Number}}(:red => [1.0, 0.0, 0.0], :yellow => [0.0, 1.0, 0.0], :blue => [0.0, 0.0, 1.0]), BCIInterface.RawDataDescriptor(8, 8))

Now you can create a Model (neural network) by using create_model:

model = create_model(modeldata)
BCIInterface.Model(Chain(Conv((5,), 200 => 64, relu, pad=2), Conv((5,), 64 => 64, relu, pad=2), Conv((5,), 64 => 128, relu, pad=2), Conv((5,), 128 => 256, relu, pad=2), Conv((5,), 256 => 512, relu, pad=2), Conv((3,), 512 => 3), flatten, Dense(18 => 3)), 0, "models/", "Standard_2023-02-05_15-07-05")

To apply this model to some inputs:

# take first 100 samples from modeldata
sample = modeldata.dataloader.data.inputs[:, :, 1:100]
# calculate results, should be 3x100 (3 outputs, 100 samples)
model(sample)

To train the model, use train!:

train!(model, modeldata, epochs=5)
